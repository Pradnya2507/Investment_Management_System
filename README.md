# Investment Management

A modern Investment Tracker web application built using React and Material UI that allows users to manage and track their investments efficiently.

---

## Features

- Add new investments (Mutual Fund, Equity, Debt, ETF)
- Track purchase date
- Enter investment amount and current value
- Clean and responsive UI using Material UI
-  Modal-based form for better user experience

---

## Tech Stack

- React.js
- Material UI (MUI)
- avaScript (ES6)
- CSS / MUI sx styling
- Bootstrap
